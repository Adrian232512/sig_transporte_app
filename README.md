# Sistema Integrado de Gestión y Monitoreo de Transporte Público

Proyecto académico de clase enfocado en el desarrollo de una aplicación móvil multiplataforma para el monitoreo de rutas de transporte en la ciudad de Cochabamba.

## Tecnologías Utilizadas
* Framework: Flutter
* Lenguaje: Dart

## Estructura de Desarrollo
El proyecto se centrará en la maquetación de las interfaces de usuario (UI) para la aplicación móvil de los pasajeros, incluyendo módulos de identificación, billetera virtual y monitoreo de unidades.
